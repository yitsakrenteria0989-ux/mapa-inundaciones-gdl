¿Qué muestra este mapa?


El mapa muestra la susceptibilidad espacial a inundaciones en el Área Metropolitana de Guadalajara, estimada mediante un modelo de aprendizaje profundo entrenado con registros históricos de inundaciones y variables geoespaciales.


Las zonas en rojo presentan las condiciones físicas más favorables para que ocurran inundaciones (pendientes bajas, alta impermeabilidad, proximidad a cuerpos de agua). Las zonas en verde son menos propensas bajo condiciones normales.


Importante: Este mapa representa susceptibilidad intrínseca del terreno, no la probabilidad de inundación ante un evento lluvioso específico.
